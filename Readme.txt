1. Запустить Main.java в командной строке с помощью maven - mvn exec:java.
2. Ввести любое число от 0 до 5, для работы с Excel файлом 11 или 12:
0 - программа завершит работу.
1 - создание файла new_file.txt, в файл записаны три слова first second third.
2 - удаление файла. Необходимо ввести имя файла, например, new_file. Файл должен существовать.
3 - переименование файла. Файл переименуется с new_file.txt на old_file.txt.
4 - поиск слова в файле. Необходмо ввести имя файла, напр., old_file. Далее ввести искомое слово, напр., first. 
Программа выводит искомое слово.
5 - замена слова в файле. Необходмо ввести имя файла, напр., old_file, далее ввести слово, которое надо переименовать (напр., second), 
далее ввести слово, на которое надо переименовать (напр., first). В результате выполнения в файл записано first first third. 
11 - создание файла MyFirstExcel.xlsx, в файл уже записаны данные.
12 - чтение из файла MyFirstExcel.xlsx, файл должен существовать.
